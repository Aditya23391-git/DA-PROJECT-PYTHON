#  Dengue Outbreak Analysis

## 📌 Overview

This project performs an exploratory data analysis (EDA) on dengue cases in India and examines the relationship between dengue outbreaks and rainfall patterns.

The objective is to identify trends, high-risk regions, and potential correlations between environmental factors and disease spread.

---

## 📂 Dataset

The project uses two datasets:

* **Dengue Cases Dataset** – Contains state-wise dengue cases and deaths
* **Rainfall Dataset** – Contains daily/state-level rainfall data

---

## ⚙️ Tech Stack

* Python
* Pandas – Data cleaning and manipulation
* Matplotlib & Seaborn – Data visualization

---

## 🔧 Data Processing Steps

* Cleaned and preprocessed both datasets
* Handled missing values
* Standardized column names
* Extracted **Year** from date columns
* Aggregated data (year-wise and state-wise)
* Merged dengue and rainfall datasets

---

## 📊 Analysis & Visualizations

* Year-wise trend of dengue cases
* Rainfall trend analysis
* Top 10 worst-affected states
* Rainfall vs dengue cases (scatter analysis)
* Correlation heatmap between key variables

---

## 📈 Key Insights

* Certain states consistently report higher dengue cases
* Years with higher rainfall often show increased dengue cases
* Positive correlation observed between rainfall and dengue spread

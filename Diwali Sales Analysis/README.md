📊 Diwali Sales Analysis
Exploratory Data Analysis using Python
Transforming raw sales data into actionable business insights
🚀 Overview

This project performs an in-depth Exploratory Data Analysis (EDA) on Diwali sales data to understand customer purchasing behavior, identify high-value segments, and uncover trends that can drive better business decisions.

It demonstrates how data can be leveraged to generate insights, strategies, and value in a real-world retail scenario.

🎯 Objectives
Analyze customer demographics (Age, Gender, Marital Status)
Identify top-performing regions and states
Evaluate product category performance
Detect high-revenue customer segments
Generate actionable business insights
🛠️ Tech Stack
Python
Pandas, NumPy → Data Analysis
Matplotlib, Seaborn → Visualization
Jupyter Notebook → Development Environment
